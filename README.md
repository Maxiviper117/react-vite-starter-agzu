# React + Vite

This template provides a minimal setup to get React working in Vite with HMR, TypeScript, and Oxlint.

## Scripts

- `npm run dev` - start the Vite dev server
- `npm run build` - type-check and build for production
- `npm run lint` - run Oxlint
- `npm run lint:fix` - run Oxlint with automatic fixes where available
